HTML(Hyper Text Markup Language) is the standard language for creting web-pages. Every web page you surf on your browser is built using HTML. The primary use of HTML is to provide a proper structure to the web page so all the content or data of the page could represent adequately. A stand-alone HTML can create only static and skeleton looking black and white pages, but with the help of CSS and Javascript,we can create a more interactive and intuitive web page. When we try to visit a website or click on the link, we basically request the server to show us the page, then the server acts on our request and sends us the appropriate HTML document as a response. Then this HTML document parsed by browse, and we are able to see the content.

Elements

Is a unit of content in an HTML document formed by HTML tags and the text or media it contains.

  • The Opening Tag: The element name used to start an HTML element. The tag type is surrounded by opening and closing angle brackets
  • The Content: The information (text or other elements) contained between the opening and closing tags of an HTML element.
  • The Closing Tag: The second HTML tag used to end an HTML element. Closing tags have a forward slash (/) inside of them, directly after the left angle bracket.

    • In HTML Formatting we have many special elements that can provide special meaning to text content. The HTML elements provide the semantic meaning to the web-page content. We usually interchangeably use the term HTML elements and tags, but technically both are different. An HTML tag is just a character inside the angled brackets. Whereas the HTML element is a collection of starting tag, its attribute, content and end tag.

    <b> = Bold the text
    <strong> = An alternative for <b> which tell that this text is essential.
    <i> = italic the text
    <em> = Similar to italic but used when you want to emphasize the text.
    <mark> = It marks the text with a default 'yellow' background colour
    <small> = Decrease the text size
    <del> = It prints a cross line over text
    <ins> = Represents the inserted text by putting an underline
    <sub> = This element is used to display the subscript
    <sup> = It can make a text superscript

    This is the Basic Code for starting a HTML Code

    <!DOCTYPE html>
    <html>
    <head>
    <title>Page Title</title>
    </head>
    <body>

    <h1>This is a Heading</h1>
    <p>This is a paragraph</p>

    </body>
    </html>

    Note: Tags in HTML are case-insensitive, i.e. they can be written in uppercase or lowercase. For example, a <title> tag could be written as <title>, <TITLE>, <Title>, <TiTlE>, etc., and it will work fine. Best practice, however, is to write all tags in lowercase for consistency, readability, and other reasons.

    Install Wsl and Apache2

    To install Wsl go to either powershell or Command Prompt. You can also install it in microsoft stor.

    In Powershell Type the following:

    wsl --install

    After Installing Wsl you can now proceed to install Apache2 in your linux subsytem type the following below:

    sudo apt-get update -y

    sudo apt-get install apache2 -y


    After installing you can now check if it is running or not by typing:

    sudo sytemctl status apache2

    sudo systmctl start apache2
